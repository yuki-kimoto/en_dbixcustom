DBIx::Custom - Mojoliciousと使うPerlのO/Rマッパー

DBIx::Customは、Mojoliciousと組み合わせて使えるPerlのO/Rマッパーです。O/Rマッパーとは、データベースへのアクセスを簡単にしてくれるモジュールのことを言います。 DBIx::Customを使うと、データベースのテーブルに対して、select、insert、update,deleteの操作が簡単にできます。 検索が柔軟にできるwhere句の機能を提供。 join句を定義して、外部キーで、テーブルを連結することも簡単です。 SQLに近い形で、構文が書けるので、純粋なSQLを使いたくなったときも、うまく対応できます。 トランザクションをサポート。 MySQL, PostgreSQL, SQLite, Oracle Database, SQL Server, Access対応。 2020年現在、廃止予定になっている機能を除いて、将来的な後方互換性を維持。

Create web site

Create web site by new command. 
giblog new mysite
Other prototypes is also available. 
giblog new_website mysite
giblog new_blog mysite
After creating web site, change directory to mysite 
cd mysite

Add new entry

Add new entry by add command. 
giblog add

Build

Build web site. 
giblog build
If you want to customize build process, you open lib/Giblog/Command/Build.pm and edit run subroutine.

Serve

You can serve your web site by morbo command if you have Mojolicious. 
morbo serve.pl
build command is automatically executed after editting files in templates directory.
